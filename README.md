# AI Programming Skills Repository

This repository contains specialized programming skills designed to be loaded by AI models (like opencode) to improve their performance in specific programming languages, frameworks, or tasks.

## Structure

Each skill is organized into its own directory containing at minimum a `SKILL.md` file.

- `python-expert/` - Python development, testing, and best practices.
- `typescript-expert/` - TypeScript, Node.js, and frontend frameworks.
- `rust-expert/` - Rust programming, Cargo, and memory safety.
- `go-expert/` - Go (Golang) development, concurrency, and tooling.

## Creating a new skill

To create a new skill, simply create a new folder and add a `SKILL.md` file:

```bash
mkdir new-skill
touch new-skill/SKILL.md
```

The `SKILL.md` should contain instructions, workflows, and context for the AI model when that skill is invoked.
